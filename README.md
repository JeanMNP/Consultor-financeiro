# 💰 Consultor Financeiro

Um aplicativo simples em **Python** usando [Dear PyGui](https://github.com/hoffstadt/DearPyGui) para ajudar no controle financeiro pessoal.  
Ele permite calcular fluxo de caixa, planejar quitação de dívidas, sugerir redução de gastos e simular poupança com juros compostos.  
Além disso, gera relatórios em **CSV** e apresenta gráficos de pizza para visualização dos gastos.

---

## 🚀 Funcionalidades
- Inserir salário e despesas (comida, contas, lazer, emergência, etc.)
- Calcular:
  - **Tempo para quitar dívidas** (com ou sem juros)
  - **Sugestões de redução de gastos**
  - **Planejamento de poupança** com meta e prazo
- Exportar resultados para **CSV**
- Visualizar **gráfico de pizza** dos gastos e fluxo de caixa
- Interface gráfica amigável com **Dear PyGui**
